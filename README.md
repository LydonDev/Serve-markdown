## Getting Started

1. Clone this repository:

   ```bash
   git clone https://github.com/LydonDev/random-rants.git
   cd random-rants
   ```

2. Install dependencies:

   ```bash
   bun install
   ```

3. Start the app:

   ```bash
   bun run dev
   ```
